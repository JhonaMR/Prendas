🎯 TIPOS DE ALERTAS
═══════════════════════════════════════════════════════════════════════════════

🟢 SUCCESS (Verde)
   • Backup creado correctamente
   • Validación exitosa
   • Imágenes respaldadas

🟡 WARNING (Amarillo)
   • Backup creado pero con problemas
   • Tablas faltantes
   • Estructura incompleta

🔴 ERROR (Rojo)
   • Error en backup
   • Corrupción detectada
   • Fallo en validación

🔵 INFO (Azul)
   • Sin imágenes para respaldar
   • Información general
